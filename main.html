Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM using JavaScript:1. By ID (getElementById),2. By Class Name (getElementsByClassName),3. By Tag Name (getElementsByTagName),4. By Name Attribute (getElementsByName),5. By CSS Selectors (querySelector and querySelectorAll) etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Property Definition Behavior Performance Security Risk innerHTML Returns or sets the HTML content inside an element, including any child elements. Parses and renders HTML tags. Slower (parses HTML). High (can be used for XSS attacks). innerText Returns or sets the visible text of an element. Ignores hidden elements and trims extra spaces. Moderate (reads computed styles). Safe (does not interpret HTML). textContent Returns or sets the raw text content, including hidden elements. Does not interpret or modify whitespace. Faster (directly reads node data). Safe (does not interpret HTML).

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements, even if they are added dynamically. Instead of adding event listeners to multiple child elements, you attach a single event listener to a common ancestor.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event starts at the target element and propagates (bubbles up) to its ancestors.
